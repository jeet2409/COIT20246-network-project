# COIT20246 Networking and Cyber Security Project

## Project Overview
This project involves the design of a secure and scalable computer network for Truelec, an electrical contracting organisation operating across Australia.

The project focuses on:
- Designing a headquarters and branch office network
- Planning IP addressing and wireless access
- Considering redundancy and reliability
- Documenting the design using GitHub

This repository contains all documentation, diagrams, and supporting files for the project.

## Repository Structure
- `README.md` – Project overview
- `plan.md` – Project planning and task management
- `network.md` – Network design and diagrams
- `diagrams/` – Network diagrams used in the design
